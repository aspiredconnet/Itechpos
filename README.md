# Itech POS Landing Page

A modern, high-performance, one-page website for Itech POS.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Custom properties (Variables), Flexbox, Grid, Animations.
- **JavaScript (Vanilla)**: DOM manipulation, Intersection Observer API.

## Features
- **Responsive Design**: Mobile-first approach.
- **Dark Mode UI**: Sleek black background with Blue (`#0070f3`) and Yellow (`#FFC000`) accents.
- **Interactivity**: Smooth scrolling, mobile menu, scroll animations, number counter.
- **SEO Optimized**: Meta tags, Open Graph tags, semantic hierarchy.

## Setup
Simply open `index.html` in any modern web browser.

For development, you can serve it using a local server:
```bash
npx serve .
```

## Folder Structure
- `index.html`: Main entry point.
- `style.css`: All styles.
- `script.js`: Functionality.
- `assets/`: Images and static resources.
